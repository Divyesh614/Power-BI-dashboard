# Power BI Dashboard Project

## Project Structure
- data/sample_sales.csv
- docs/dashboard_requirements.md
- powerbi/README_PBI.md

## Steps
1. Open Power BI Desktop.
2. Import data/sample_sales.csv.
3. Create visuals and save as Dashboard.pbix.
4. Push this project to GitHub.

This folder is VS Code friendly and can be version-controlled in Git.
